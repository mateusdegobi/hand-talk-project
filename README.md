# HandTalk Project

## Rodando o app
1. Clone esse repositório:

```bash
git clone https://github.com/mateusdegobi/hand-talk-project.git
```

2. Entre na pasta do projeto e instale as dependências

```bash
  cd desafio-hand-talk && yarn
```

3. Inicie o app

```bash
 yarn start | npx expo start
```
